## NovaStar Controller
This module will allow you to control the following NovaStar products: MCTRL4K, VX4S, VX6S, NovaProHD, NovaPro UHD Jr,VX1000, VX600, VX16s, J6

### Configuration
* Enter the IP address of the device in the configuration settings.
* The device will use TCP port 5200.

**Available actions:**
* Change Brightness
* Change Test Patterns
* Change Display Mode
* Change Inputs

**VX4S, VX6S, NovaProHD, NovaProHDJr only:**
* PIP On/Off
* Change Scaling
* Recall Presets
